# Currículo Online Responsivo

Aplicação web de currículo online desenvolvida com **HTML, CSS e JavaScript**, com layout responsivo e funcionalidade de geração de PDF para download.

# Funcionalidades

-  Layout totalmente responsivo (mobile e desktop)
-  Geração de currículo em PDF
-  Seções organizadas:
  - Resumo profissional
  - Formação acadêmica
  - Stacks / Tecnologias
  - Certificados e cursos
  - Projetos
-  Links para redes profissionais (LinkedIn e GitHub)

---

# Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- html2pdf.js
